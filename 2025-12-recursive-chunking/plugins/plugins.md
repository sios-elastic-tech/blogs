# Plugins

このサンプルを動かすために必要なプラグイン。

## analysis-icu

参考URL: https://www.elastic.co/docs/reference/elasticsearch/plugins/analysis-icu

```
sudo bin/elasticsearch-plugin install analysis-icu
```

## analysis-kuromoji

参考URL: https://www.elastic.co/docs/reference/elasticsearch/plugins/analysis-kuromoji

```
sudo bin/elasticsearch-plugin install analysis-kuromoji
```

## restart

Elasticsearch node を再起動する。
